Let's get technical

Let’s talk about starting projects.

Is there anything harder, more oppressive in the creative process than searching for that initial inspiration? The trope has existed for as long as stories have been told, through images of writers sat in front of blank pieces of paper, artists with empty canvasses, coders with empty git repos… Okay, maybe I made the last one up. We coders get it a little easier with regards to getting started, with a well trodden path to follow.

git init

Programmers never have to deal with the tabula rasa of an empty project directory. Modern best practise means that everyone uses source control, by which of course I mean git. And therefore, even from the soulless depths of an empty directory shines the beacon of hope…
